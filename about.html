About Me

Create an array with nested objects (at least four nested objects) having the following properties firstname, last name, likes, number with respective values of your choice for each.

Create a function that takes in two parameters to represent first name and any other object key of your choice. The function should check if the first name passed to the parameters exixts for the nested objects. If it does exist, it should return the other object key value of your choice.
Requirements
Your program should be able to handle cases where the values passed is not in the object created. ie displaying error messages(firstname or likes does not exist)
Your outputs should be console logged
Hint
Use loop to iterate through the object created.
Think about condition statements for handling error cases and checking if the first name and any other key value passed exists in the object created.
